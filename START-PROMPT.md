# Speckle Skill Package — START PROMPT

## Universele Start Prompt

Gebruik deze prompt bij het starten van een nieuwe sessie:

---

> Lees de ROADMAP.md en ga verder waar we gebleven zijn. Volg de 7-fase methodologie uit WAY_OF_WORK.md.

---

## Wat deze prompt doet

1. **P-001 Session Start** wordt getriggerd — alle governance bestanden worden gelezen
2. **ROADMAP.md** bepaalt de huidige fase en voortgang
3. **WAY_OF_WORK.md** definieert de methodologie voor de volgende stap
4. **LESSONS.md** voorkomt bekende fouten
5. **DECISIONS.md** geeft architecturele context

## Alternatieve Start Prompts

> "Start fase B1 (Raw Masterplan). Lees eerst ROADMAP.md en WAY_OF_WORK.md."

### Skill creation sessie
> "Start skill creation batch. Lees ROADMAP.md voor de huidige batch en volg P-002 en P-003."

### Validatie sessie
> "Valideer alle geschreven skills. Volg P-003 (Quality Control Protocol)."
